# AI Resume Screening System

An AI-powered Applicant Tracking System (ATS) built using FastAPI, React, PostgreSQL, and Machine Learning.

## Features

- Resume Upload
- PDF Parsing
- ATS Score Calculation
- Candidate Ranking
- Recruiter Dashboard
- JWT Authentication
- REST APIs
- Docker Support

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Sentence Transformers
- React
- Docker
